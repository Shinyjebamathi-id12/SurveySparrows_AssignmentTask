SurveySparrows Assignment - Google Calendar Clone (Month view)
===============================================================

What you get
- Responsive React app (month view) that loads events from a static JSON.
- Current date highlight, previous/next month navigation.
- Multiple events on the same day displayed with simple conflict coloring.
- Attractive gradient background and modern card UI.
- Works on phone, tablet, and desktop.

How to run
1. unzip or clone this folder.
2. In the project root run:
   npm install
   npm start
3. App will open at http://localhost:3000
